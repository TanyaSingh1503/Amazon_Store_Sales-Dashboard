# Amazon_Store_Sales-Dashboard
This dashboard provides a clear analysis of Amazon sales performance through key metrics such as total sales, category-wise sales, and product trends. Interactive visualisations help identify growth patterns, customer behaviour, and high-performing segments to support better business decisions.

Tech Stack: Excel, Power BI, DAX
